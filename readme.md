# Gold Check

Gold Check is a cron job for GitHub Actions that will notify through email with daily gold price from [Brankas](https://www.brankaslm.com/antam/index) website.

## Requirements
- Python
- Selenium
- BeautifulSoup
- ChromDriver Auto Installer
- PyVirtualDisplay
